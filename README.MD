If you are looking for the specific artifacts used in our incident investigation extraction models, please navigate using the quick links below:

### Navigation

*   **[IBAMA Prompts](./artefatos/prompts/ibama/)**: Contains the text files used for agent instructions (e.g., extracting phenomena, causes and consequences).
*   **[Raw Dataset](./artefatos/raw/)**: Contains the original, unprocessed incident communication data from IBAMA.
*   **[NER Dataset](./artefatos/ner_datasets/)**: Contains the processed Named Entity Recognition datasets that we use for our ground truth.
